# Game Projects

This folder contains a collection of game development projects created as part of my learning journey in game programming.

## Overview

These projects were developed to practice game development concepts, including gameplay mechanics, object interactions, animations, and user interface design.

## Projects

- **Objects Game** – A game focused on object interactions and gameplay mechanics.
- **Space Game** – A space-themed game featuring player movement, enemies, and interactive gameplay.

## Technologies

- JavaScript
- HTML5
- CSS3
- Phaser.js

## Purpose

The purpose of these projects is to improve my game development skills, explore different game mechanics, and gain hands-on experience using the Phaser framework.
