This folder contains a collection of web pages developed as part of the **Web Programming and Designing** course.

The projects in this folder were created to practice front-end web development concepts, including page layout, user interface design, and responsive web page development using standard web technologies.

- Contact the Team Page

## Technologies

- HTML5
- CSS3
- JavaScript

## Purpose

The purpose of these exercises is to strengthen my web development skills by applying front-end design principles and building functional, user-friendly web pages as part of academic coursework.
