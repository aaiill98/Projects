# Taif Health Cluster Training Platform

This project was developed as part of a university course to support the training process at Taif Health Cluster. The platform provides an organized system for managing training applications submitted by students and other individuals seeking training opportunities.

## Overview

The platform enables applicants to register, submit training requests, and track their application status. It also provides administrative interfaces for managing applications, reviewing requests, and organizing the training workflow.

## Features

- User registration and login
- Submit training applications
- Track application status
- Administrative dashboard
- Department management
- Queue management for applications
- Database integration using PHP and MySQL

## Technologies

- PHP
- HTML5
- CSS3
- JavaScript
- MySQL

## Project Structure

- `index.php` – Home page
- `login.php` – User login
- `signup.php` – User registration
- `student_apply.php` – Training application form
- `student_status.php` – Application status
- `manager.php` – Manager dashboard
- `admin.php` – Administration panel
- `connection.php` – Database connection
- `taif_slim.sql` – Database schema

## Purpose

The purpose of this project is to simplify the training application process by providing a centralized platform for applicants and administrators, improving application management and communication throughout the training process.
