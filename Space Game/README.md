# Space Game

Space Game is a 2D arcade game developed using Phaser.js as part of my game development learning journey.

## Overview

In this game, the player navigates through a space-themed environment, collects stars to increase the score, and avoids enemies. Touching an enemy results in a game over, making timing and movement essential to success.

## Features

- 2D arcade gameplay
- Collect stars to earn points
- Avoid enemies to survive
- Collision detection
- Score system
- Simple and responsive controls

## Technologies

- JavaScript
- HTML5
- CSS3
- Phaser.js

## Purpose

This project was created to practice game development concepts, including player movement, collision detection, enemy behavior, score tracking, and interactive gameplay using the Phaser.js framework.
