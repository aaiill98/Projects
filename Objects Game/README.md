# Objects Game

Objects Game is a simple 2D game developed to practice game development concepts using the Phaser.js framework.

## Overview

The game focuses on player interaction with different objects while implementing core game mechanics such as movement, collision detection, and scoring.

## Features

- 2D gameplay
- Object interaction
- Collision detection
- Score system
- Simple and interactive gameplay

## Technologies

- JavaScript
- HTML5
- Phaser.js
- CSS

## Purpose

This project was created to improve my understanding of game development fundamentals and gain practical experience with the Phaser.js framework.
