# Nomu – Smart Agricultural Platform

Nomu is a smart agricultural platform developed as a graduation project to help farmers monitor their crops and make more informed harvesting decisions using artificial intelligence.

## Overview

The platform leverages AI technologies to analyze olive and fig crops through images and videos. It provides real-time fruit detection, counting, and ripeness estimation to support efficient harvesting.

## Features

- 🌱 AI-powered crop monitoring
- 🍈 Fruit detection and counting using YOLOv8
- 📊 Ripeness estimation using Support Vector Machine (SVM)
- 🖼️ Image and video analysis
- 📈 Real-time analysis results
- 🤝 Connects farmers with volunteers to organize harvesting activities

## Technologies

- Python
- Flask
- YOLOv8
- Support Vector Machine (SVM)
- HTML
- CSS
- JavaScript

## Supported Crops

- Olive
- Fig

## Project Purpose

Nomu aims to improve agricultural productivity by combining artificial intelligence with a user-friendly platform that helps farmers monitor crop conditions, estimate harvest readiness, and coordinate harvesting with volunteers.
